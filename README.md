# OnChainRisk

**Blockchain forensics platform for crypto wallet analysis, risk scoring, and transaction tracing.**

Analyze wallets, trace fund flows, detect scams, and generate court-ready evidence across 23 blockchain networks.

## What is OnChainRisk?

OnChainRisk is a blockchain forensics platform that helps investigators, compliance teams, and individuals assess the risk of any crypto wallet address.

It combines transaction analysis, pattern detection, entity labeling, and AI-powered investigation into a single tool — accessible at a fraction of the cost of enterprise solutions.

Supports 23 blockchain networks including Ethereum, Bitcoin, Solana, and Layer 2 ecosystems.

## Key Features

### Wallet Risk Scoring
Analyze any crypto wallet address and get a 0-100 risk score based on transaction patterns, counterparty risk, sanctions exposure, and behavioral signals.

### Fund Flow Tracing
Interactive graph visualization of fund movements between wallets. Expand nodes, trace paths across multiple hops, and identify where funds end up.

### Multi-Chain Wallet Scan
Scan one address across 15 EVM networks simultaneously. Get a composite risk score with per-network breakdown.

### AI Investigation Agent
AI-powered investigation agent for automated wallet analysis and fund tracing. Analyzes addresses, detects patterns, and builds evidence through natural language conversation.

### Block MEV Analysis
Analyze any block for sandwich attacks, arbitrage, flash loans, and transaction ordering manipulation across 10 EVM networks.

### Token Security Scoring
Check any token for honeypot risk, buy/sell tax simulation, ownership concentration, and liquidity analysis.

### Scam Detection
Detect common scam patterns including pig butchering, address poisoning, rug pulls, and mixer interactions.

### Court-Ready Reports
Export analysis as PDF, CSV, XLSX, or Markdown. Reports include fund flow diagrams, risk flags, counterparty analysis, and evidence chains.
